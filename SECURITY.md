# Security Policy

## Supported Versions

Only the latest version of **OpenCEP Ultra** is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | ✅ Yes             |
| Original OpenCEP | ❌ No (Upstream) |

## Reporting a Vulnerability

We take the security of our project seriously. If you find a vulnerability, please do NOT open a public issue. Instead, follow these steps:

1.  Email a detailed report to **contato@kedllon.solutions**.
2.  Include a description of the vulnerability and steps to reproduce it.
3.  Allow us up to **7 days** to verify and respond before sharing details publicly.

Thank you for helping keep OpenCEP Ultra safe for everyone!
