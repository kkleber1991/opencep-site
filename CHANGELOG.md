# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-09

### Added
- Initial release of **OpenCEP Ultra**.
- Jamstack architecture (Cloudflare R2 + Workers).
- Premium landing page with Dark Mode and Glassmorphism.
- Support for consolidated state, city, and neighborhood filters.
- Intelligent 404 routing (JSON for API, HTML for humans).
- Support for Google Analytics and Microsoft Clarity.
