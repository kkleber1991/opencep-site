A API mais completa e resiliente

OpenCEP Ultra

Uma API de CEP de alta performance construída sobre arquitetura estática (Jamstack). Sem bancos de dados lentos, apenas arquivos super-rápidos entregues via CDN.

Ver Documentação

🚀 Ultra Velocidade

Graças ao Cloudflare R2 e cache global, os dados são entregues em milissegundos, independente de onde o usuário esteja.

📍 Filtros Avançados

O grande diferencial: você pode buscar listas completas de CEPs filtrando por Estado, Cidade ou Bairro com uma única chamada.

💎 100% Gratuito

Nossa missão é democratizar o acesso aos dados postais brasileiros. Sem limites abusivos ou custos ocultos.

Integrando a API

1. Consulta por CEP Individual

Obtenha os dados completos de um CEP específico.

GET https://api-cep.kedllon.solutions/v1/01001000.json
2. Lista de CEPs por Cidade

Recupere todos os CEPs de uma cidade específica de uma só vez.

GET https://api-cep.kedllon.solutions/cidades/SP_Sao_Paulo.json
3. Lista de CEPs por Bairro

A maior profundidade de dados disponível de forma estática.

GET https://api-cep.kedllon.solutions/bairros/SP_Sao_Paulo_Se.json
4. Lista de CEPs por Estado

Todos os códigos postais de um estado brasileiro.

GET https://api-cep.kedllon.solutions/estados/AC.json
